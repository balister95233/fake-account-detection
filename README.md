## 🔍 Fake Social Media Account Detection

## 📌 Description
This project uses Machine Learning to detect fake social media accounts based on user data and behavior.


## 🚀 Features
- Detects fake vs real accounts
- Uses ML algorithms for prediction
- Simple and easy to use


## 🛠️ Tech Stack
- Python
- Machine Learning
- Pandas, NumPy, Scikit-learn


## 📊 How it Works
- Collect dataset
- Train ML model
- Predict whether account is fake or real


## ▶️ How to Run
1. Clone the repository  
2. Install dependencies  
3. Run the Python file
